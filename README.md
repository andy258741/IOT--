# IOT--Pet-Feeding-Machine
## 寵物餵食機 
這個作品利用rasbperry pi和簡單的電子零件，搭配上linebot，可以選擇時間與餵食的量，在手機上就能輕鬆地幫家中的寵物餵食
## 硬體
> #### 樹梅派3(Raspberry pi 3)  
> #### 紅外線感測器(pir motion sensor)  
> #### 相機模組(camera module)  
> #### 減速馬達(GA12-N20)
> #### 厚紙板(cardboard)
